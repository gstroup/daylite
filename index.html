daylite

Overview

This little UI widget is intended to be lightweight and fast for use on mobile browsers as well as desktop browsers. It is written as a requirejs module, and also depends on the moment.js library for date manipulation. But there are no other dependencies on any JS frameworks.

Usage

First call the daylite() constructor method, passing in the id of a div to hold the calendar. Then call the init() method, with an options object as described here. Finally, call the open() method when you want to see the calendar.
